# Linux-cfg

## 基本介绍

这是一个linux上本人在用配置的集合,配置了自动部署脚本,欢迎大家一起优化哦

## **目录结构**

- `sync-config.sh`  自动配置脚本
- `server`  服务器相关
  - `server-backup.sh`  服务器自动备份
- `general`  通用配置
  - `.vim/` & `.vimrc`  vim 相关配置
  - `.tmux.conf`  tmux配置
  - `.bashrc`  linux基础环境配置
- `arch`  archlinux系统专用配置
  - `archlinux-autoInstall.sh`  archlinux 自动安装脚本
